See it in action

Real question, real answer — with the rules cited.

I blocked a Colossal Dreadmaw (6/6, trample) with a Typhoid Rats (1/1, deathtouch). What happens?
Verdict: the opponent takes 5 damage and both creatures die.

The Dreadmaw only needs to assign 1 point of lethal damage to its blocker (rule 702.19, Trample) — the other 5 trample through. The Rats destroys the Dreadmaw with 1 point of deathtouch damage (rules 702.2b and 704.5h).

How to use

Magic Judge speaks MCP — the open standard that plugs new abilities into AI assistants. Pick your app, paste one link, done.

Step 1 · Claude app

Add the connector

In the Claude app or website, go to Settings → Connectors → Add connector and paste this link:

https://mcp.magicjudge.app/mcp
Step 1 · Terminal

Claude Code & dev tools

One command in your terminal:

claude mcp add --transport http magic-judge \ https://mcp.magicjudge.app/mcp

The same endpoint works in Cursor, VS Code and any other MCP client.

Step 2 · Optional

Install the skills pack

Skills make the judge even sharper in Claude: rules lookup method, card search syntax and bracket-aware deck building.

Download the pack and upload it under Settings → Skills.
